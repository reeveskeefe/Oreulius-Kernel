# Licensing and Legal Documents

This folder contains the supplementary legal documents for the Oreulius project.
The primary license text lives in `LICENSE` at the repo root; the files here
provide context, detail, and operational guidance for specific audiences.

| File | Purpose |
|---|---|
| [COMMERCIAL.md](COMMERCIAL.md) | What requires a commercial license, what is permitted under the public license, and how to start a commercial licensing conversation. |
| [CONTRIBUTOR-LICENSE.md](CONTRIBUTOR-LICENSE.md) | Terms that govern the rights contributors grant to the project when submitting code or other material. You retain copyright; you grant the project a broad license to use and redistribute your contribution. |
| [LICENSE-FAQ.md](LICENSE-FAQ.md) | Plain-English answers to common questions about what is and is not permitted under the Oreulius Community License. Non-binding; the LICENSE file controls. |
| [PRIVACY.md](PRIVACY.md) | How the project handles personal data it collects (contact, issue submissions, licensing requests). Does not cover data you process using Oreulius itself. |
| [WHY-LICENSE.md](WHY-LICENSE.md) | A short explanation of the rationale behind the licensing approach — why source-available rather than fully open-source, and what the model is designed to enable. |

For any licensing question not answered here, contact **reeveskeefe@gmail.com**.
