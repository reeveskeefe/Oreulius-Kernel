# Oreulius Documentation Home

This directory is the documentation funnel for Oreulius.

Use it in this order:

1. understand what the project is
2. get one successful demo running
3. choose the subsystem you want to study more deeply

## Directory Guide

- `project/`
  - positioning, MVP scope, and product framing
- `architecture/`
  - theory, mathematical framing, assembly reference, and deeper system shape
- `capability/`
  - authority model, CapNet, policy, predictive revocation, and delegation graph material
- `ipc/`
  - IPC design and implementation roadmap
- `runtime/`
  - WASM execution, JIT hardening, ABI, and app-facing runtime guidance
- `services/`
  - service pointers, observers, polyglot runtime, and service model material
- `storage/`
  - filesystem, persistence, temporal objects, and temporal capability state
- `assets/`
  - screenshots, branding, and badge assets
